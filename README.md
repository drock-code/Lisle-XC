# Lisle High School Cross Country Web App
Web app for the Lisle High School cross country team that will be hosted at [lislexc.com](https://lislexc.com/). More modern version of the [School Sports Sites](https://sportssites.us/) platform.

The original School Sports Sites was built with a PHP backend and a jQuery/Javascript frontend. The database runs on MySQL.

The purpose of this project is to update the platform to run on Next.js while keeping the MySQL database. The database will be normalized because the original database had some inconsistencies and was not normalized.
